# Resume-Building
📄 Professional Resume Builder

A simple and modern web-based Resume Builder that allows users to create professional resumes quickly.
Users can enter their personal information, work experience, education, and skills, preview the resume live, and download it as a PDF file.

This project is built using HTML, CSS, and JavaScript and works directly in the browser without any backend.

🌐 Project Overview

The Professional Resume Builder helps users easily generate a well-structured resume.
The website provides a form interface where users input their details, and the system instantly updates the resume preview in real time.

Once the resume is ready, users can download the resume as a PDF with a single click.

This tool is useful for:

Students

Job seekers

Freelancers

Developers building portfolio projects

✨ Features
1️⃣ Live Resume Preview

As users enter information, the resume preview updates automatically.

No need to refresh the page.

2️⃣ Add Multiple Entries

Users can dynamically add:

Work Experience

Education

Skills

3️⃣ Remove Entries

Each section includes a remove button to delete unwanted items.

4️⃣ Professional Resume Layout

The resume includes:

Header with name and contact details

Professional summary

Work experience section

Education section

Skills section

5️⃣ Download Resume as PDF

The website uses html2pdf.js to generate a downloadable PDF resume.

6️⃣ Fully Responsive Design

The layout adapts to:

Desktop

Tablet

Mobile devices
